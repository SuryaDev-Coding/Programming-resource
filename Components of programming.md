# Components:
* ### Systems programming: [Low Level](https://www.youtube.com/@LowLevelTV)
  Hardware interaction with software, it is sllightly harder than others but also less competition.
  * **Operating Systems**: OS for windows, linux, macos. *C, C++, Assembly, Rust*
  * **Embedded Systems & Iot**: Cars, modems, microcontrollers . *C, C++, Rust*
  * **Compilers**: High level prgramming language to machine instruction. *C++, Rust, OCaml*
  * **Game Engine**: Not game, but app used to build games. *C++, C#(unity)*
* ### Web development: [Traversy Media](https://www.youtube.com/@traversymedia)
  Programs for internet, website, servers.
  * **Frontend**: The website that user sees. *Javascript, Typescript, HTML, CSS*
  * **Backend**: The program running on server which enables frontend. **Python, Java, Javascript/Typescript(Node.js), Go, C#, ruby, PHP(maintanance)*
  * **Full Stack**: Programmer who does both frontend and backend.
* ### Mobile app development: [2days long video, watch it over a semester, while building](https://www.youtube.com/watch?v=blKkRoZPxLc) 
  Apps for mobile phone.
  * **IOS**: Apps for apple device. *Swift, Objective-C(Old program maintance)*
  * **Android**: Apps for android. *Java, Kotlin*
  * **Cross platform**: Both android and apple. *Dart (Flutter), JavaScript/TypeScript (React Native)*
* ### Enterprise & Desktop Application Development: [Traversy Media](https://www.youtube.com/@traversymedia) And [Freecode](https://www.youtube.com/@freecodecamp)
  For big companies and windows apps.
  * **Enterprise**: Large scale program for corporations including banks. *Java(Most companies), C#(Microsoft mostly), COBOL (old bank program maintanace).*
  * **Desktop Application**: *C# (.NET), C++, Swift (macOS), Java, JavaScript (via Electron)*
* ### Data Science, AI, & Machine Learning: [Andrj Karpathy](https://www.youtube.com/@AndrejKarpathy)
  A very fast growing and hyped sector.
  * **Datascience**: Storing and recieving and transferring high volume of data. *Python, Scala, Java, SQL*
  * **AI/ML**: Artificial intelligence. *Python(Dominant, you use modules), C++(How build modules), R*
* ### Cloud Computing & DevOps: [TechWorld with Nan](https://www.youtube.com/@TechWorldwithNana/playlists)
  Bascially you build apps and deploy them too, both in one team.
  * **Cloud computing**: Apps such Github, AWS(Very important in our life), Drive. *Python, Go (Golang), and JavaScript/TypeScript*
  * **Site reliabilty**: Check health of server/System. *Go, Python, Bash/Shell scripting.*
* ## *CyberSecurity*:
  Cybersecurity cuts across every single section, you can become a cybersecurity engineer from any of the above. So find one from above, then try cyber security, if you like it, do it.

NOTE: You do not have to be well versed in all of these, you have to be in any one within next four years, but I highly suggest you to try all of these in first semester, not just one program, but a month dedicated to it. If you liked what you did, double down on it.
