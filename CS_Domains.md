# Components:
Note: Difficulty->D: High/Medium/Low and Competition->C: High/Medium/Low. Do not decide using this, this is just info.
* ### Systems programming: [Low Level](https://www.youtube.com/@LowLevelTV) D:H C:L
  Hardware interaction with software.
  * **Operating Systems**: OS for windows, linux, macos. *C, C++, Assembly, Rust*
  * **Embedded Systems & Iot**: Cars, modems, microcontrollers . *C, C++, Rust*
  * **Compilers**: High level prgramming language to machine instruction. *C++, Rust, OCaml*
  * **Game Engine**: Not game, but app used to build games. *C++, C#(unity)*

    How to get begin: Learn C -> Book CS:APP First chapter -> Project: [Unix Shell](https://www.youtube.com/watch?v=yTR00r8vBH8&t=1s)
* ### Web development: [Traversy Media](https://www.youtube.com/@traversymedia) D:M C:H (but if and/or backend)C:M-H 
  Programs for internet, website, servers.
  * **Frontend**: The website that user sees. *Javascript, Typescript, HTML, CSS*
  * **Backend**: The program running on server which enables frontend. **Python, Java, Javascript/Typescript(Node.js), Go, C#, ruby, PHP(maintanance)*
  * **Full Stack**: Programmer who does both frontend and backend.

    How to get started: Learn HTML;CSS -> Learn Javascript -> Project: [Website](https://www.youtube.com/watch?v=-u3vE84Wo_U)
* ### Mobile app development: [Complete Flutter course](https://www.youtube.com/watch?v=VPvVD8t02U8) D:M C:M
  Apps for mobile phone.
  * **IOS**: Apps for apple device. *Swift, Objective-C(Old program maintance)*
  * **Android**: Apps for android. *Java, Kotlin*
  * **Cross platform**: Both android and apple. *Dart (Flutter), JavaScript/TypeScript (React Native)*
 
    How to get started: [Choices](https://www.youtube.com/watch?v=yye7rSsiV6k) -> Learn Language you like -> Project(assuming flutter): [Language translator](https://www.youtube.com/watch?v=CeupjFXlFiQ&list=PLG5KvF1OpdCV0mGf8k7NNqFyI9qlwIiMl&index=2)
* ### Enterprise & Desktop Application Development: [Traversy Media](https://www.youtube.com/@traversymedia) And [Freecode](https://www.youtube.com/@freecodecamp) D:M C:M
  For big companies and windows apps.
  * **Enterprise**: Large scale program for corporations including banks. *Java(Most companies), C#(Microsoft mostly), COBOL (old bank program maintanace).*
  * **Desktop Application**: *C# (.NET), C++, Swift (macOS), Java, JavaScript (via Electron)*

    How to get started: Learn TypeScript/Electron -> Learn small amount of Java -> Project: [Screen Recorder](https://www.youtube.com/watch?v=3yqDxhR2XxE)
* ### Data Science, AI, & Machine Learning: [Andrj Karpathy](https://www.youtube.com/@AndrejKarpathy) D:H C:H
  A very fast growing and hyped sector.
  * **Datascience**: Storing and recieving and transferring high volume of data. *Python, Scala, Java, SQL*
  * **AI/ML**: Artificial intelligence. *Python(Dominant, you use modules), C++(How build modules), R*

    How to get started: Learn Python+Python Modules -> Project: [Machine Learning](https://www.youtube.com/watch?v=29ZQ3TDGgRQ) -> [Roadmap](https://www.youtube.com/watch?v=7IgVGSaQPaw&t=123s)
* ### Cloud Computing & DevOps: [TechWorld with Nan](https://www.youtube.com/@TechWorldwithNana/playlists) D:M-H C:M
  Bascially you build apps and automating deploy of them.
  * **Cloud computing**: Apps such Github, AWS(Very important in our life), Drive. *Python, Go (Golang), and JavaScript/TypeScript*
  * **Site reliabilty**: Check health of server/System. *Go, Python, Bash/Shell scripting.*
  
    How to get started: ** ⚠️Disclaimir: There is NO entry level job for DevOps, it is a senior level job. I warn you about clickbait youtube videos which promise 20 lakhs job for DevOps, it exists but not for entry level. I higly suggest doing from above.** But here what you need to learn is Linux and Github. [RoadMap](https://www.youtube.com/watch?v=6GQRb4fGvtk)
    
* ## *CyberSecurity*: D:H C:M(L high skill)

Learn this in third semester. [Roadmap](https://www.youtube.com/watch?v=v7BNtpw53AA) -> Resource: [TryHackMe](https://tryhackme.com/)
