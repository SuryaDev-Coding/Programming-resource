# Programmin Resource:
## Learning a concpet:
* [TeachYourselfCS](https://teachyourselfcs.com/): A website which contains concpts and required books and video resource.
## Learning A language:
*  Python tutorial: [English](https://www.youtube.com/results?search_query=python+tutorial) | [Tamil](https://www.youtube.com/watch?v=m67-bOpOoPU)One of the best language to start learning.
* [codechef](https://www.codechef.com/): The website that I use for learning a language.
* [Exercism](https://exercism.org/): A easier version of [Leetcode](https://leetcode.com/)(This is what used by companies for hiring). This is     slightly harder. So I recommend this order: CodeChef -> Exercism -> Leetcode.
* [Codewars](https://www.codewars.com/): This is the gamified version to learn a programming language.
* [w3schools](https://www.w3schools.com/): This is a programming tutorial that is good for web development.
* [Jiki](https://jiki.io): A website fom exercism for learning programming.
