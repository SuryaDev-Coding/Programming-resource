## Learning A language: 
**To learn a language, use this order**
*  Python tutorial: [English](https://www.youtube.com/results?search_query=python+tutorial) | [Tamil](https://www.youtube.com/watch?v=m67-bOpOoPU) Use this to learn basics and syntax, around one hour of this is enough.
* [Codechef](https://www.codechef.com/): Use this to practise beginner level programs.
* [**Exercism**](https://exercism.org/):  If you want only one website, this is one the I recommend. After learning basics, use this to learn to write idiomatic code and programming fluency.
* [Leetcode](https://leetcode.com/): Once you are comfortable with programming, start this. [Neetcode](https://www.youtube.com/@NeetCode/playlists) For help.
* [W3Schools](https://www.w3schools.com/): This is a tutorial webiste which is good for web development.
* [Jiki](https://jiki.io): A website for learning begineer programming *concept*.
